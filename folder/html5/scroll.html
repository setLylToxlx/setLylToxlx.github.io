script

                
                    window.addEventListener('scroll', function() {
                        console.log('this window')
                    })
                
                    document.addEventListener('scroll', function() {
                        console.log('this document')
                    })
                    document.body.addEventListener('scroll', function() {
                        console.log('this body')
                    })
                    document.getElementById('app').addEventListener('scroll', function() {
                        console.log('this div')
                    })

html, body 设置 {height: 100%; overflow-y/overflow: auto/scroll}, window,document的scroll事件不起作用；

html, body 没设置 height: 100%; 或者 overflow-y/overflow: auto/scroll任意一项, window,document的scroll事件起作用；document.body的scroll事件不起作用

div 设置{height: 100vh; overflow-y/overflow: auto/scroll}, div 的scroll事件起作用，window、document、document.body三者失效

div设置可行、document.body也可行时，div的起效
